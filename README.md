# Datos abiertos (datos-madrid-es)

Datos abiertos is a open data portal open-data portal for Spain running **CKAN**, exposing a standardized catalog API over ~667 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/datos-madrid-es/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=datos-madrid-es-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN  ·  **version:** 2.9.11

## API
- **Datos abiertos CKAN Action API** — base `https://datos.madrid.es/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://datos.madrid.es)
- Live endpoint verified: `https://datos.madrid.es/api/3/action/status_show` · ~667 datasets
